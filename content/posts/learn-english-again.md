## 二、语言习得理论支持

### 1. 输入假说（Input Hypothesis）- Krashen

**核心观点**：

> 语言习得发生在学习者接收"可理解输入"（i+1）时——即略高于当前水平的语言输入。

**本计划的应用**：

- Phase 1-2 大量阅读英文内容 → 提供可理解输入
- 阅读 Web3 相关内容 → 内容相关、有意义
- 从简单到复杂逐步升级 → 符合 i+1 原则

**理论支持来源**:

- Krashen, S. (1985). The Input Hypothesis: Issues and Implications
- Krashen, S. (1982). Principles and Practice in Second Language Acquisition

------

### 2. 输出假说（Output Hypothesis）- Swain

**核心观点**：

> 仅有输入不够，学习者需要"被推动"产出语言（pushed output），这有三个功能：
>
> 1. **注意功能**：产出时注意到自己的语言缺陷
> 2. **假设检验功能**：尝试新的语言形式并获得反馈
> 3. **元语言功能**：反思语言使用，内化知识

**本计划的应用**：

- 每日写作练习 → 大量输出
- 错误记录和纠正 → 注意功能
- 发推特获得反馈 → 假设检验
- 总结和复盘 → 元语言反思

**理论支持来源**:

- Swain, M. (1985). Communicative Competence: Some Roles of Comprehensible Input and Comprehensible Output in Its Development
- Swain, M. (1995). Three Functions of Output in Second Language Learning

------

### 3. 交互假说（Interaction Hypothesis）- Long

**核心观点**：

> 与他人互动时的"意义协商"（negotiation of meaning）促进语言习得，因为互动提供：
>
> - 可理解输入
> - 反馈
> - 修正机会

**本计划的应用**：

- Phase 3-4 与语伴/外教对话 → 真实互动
- Twitter 互动和评论 → 书面互动
- Twitter Spaces 参与 → 口语互动

**理论支持来源**:

- Long, M. H. (1996). The Role of the Linguistic Environment in Second Language Acquisition

------

### 4. 技能习得理论（Skill Acquisition Theory）

**核心观点**：

> 语言学习经历三个阶段：
>
> 1. **认知阶段**：学习规则（知道）
> 2. **联结阶段**：练习应用（能做但慢）
> 3. **自动化阶段**：熟练运用（自动化）

**本计划的应用**：

| 阶段   | 本计划对应         | 时间        |
| ------ | ------------------ | ----------- |
| 认知   | Phase 1 语法学习   | Month 1-3   |
| 联结   | Phase 2-3 练习应用 | Month 4-9   |
| 自动化 | Phase 4-5 实战使用 | Month 10-18 |

**理论支持来源**:

- DeKeyser, R. (2007). Skill Acquisition Theory

## 三、实证研究支持

### 研究 1：沉浸式学习 vs 课堂学习

**研究发现**（Swain, 1985）：

- 加拿大法语沉浸项目的学生
- 听力和阅读能力接近母语者
- 但口语和写作落后
- 原因：输入充足，但输出机会不足

**本计划的借鉴**：

- 不只是输入（阅读/听力）
- 强调输出（写作/口语）
- 逐步增加输出比例

------

### 研究 2：输出与语法习得

**研究发现**（Nobuyoshi & Ellis, 1993）：

> 被"推动"产出正确语法的学习者，不仅即时表现提升，长期准确度也显著提高。

**本计划的借鉴**：

- 每日写作纠错
- 记录错误并复习
- 被"推动"写出正确的英语

------

### 研究 3：成人语言学习能力

**研究发现**：

> 成人学习能力随年龄下降，但学习潜力持续存在。成人可以通过策略性学习弥补生理劣势。

**本计划的借鉴**：

- 系统化学习计划
- 利用分析能力（语法规则学习）
- 目标导向的学习

**来源**: Wang (2023), 引用于 The Impact of Krashen Input Hypothesis

## 四、为什么"先读写，后听说"可行

### 传统顺序 vs 本计划顺序

| 传统（母语习得）  | 本计划（成人外语） |
| ----------------- | ------------------ |
| 听 → 说 → 读 → 写 | 读 → 写 → 听 → 说  |
| 需要大量时间沉浸  | 时间有限，按需学习 |
| 无读写基础        | 已有读写能力       |
| 5-7 年达到流利    | 12-18 个月达到 B2  |

### 为什么这样做有效

### Nation 的研究支持

**研究发现**（Nation, 引用于 OptiLingo）：

> 通过阅读可理解输入，学习者可以在约 180 小时内达到相当于 B2 的阅读水平。

这表明：专注阅读可以大幅加速某一技能的发展，然后再扩展到其他技能。
