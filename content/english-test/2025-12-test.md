## Part 1: Writing 详情

### 1A. 短写作（Twitter 风格）

**题目：** 你发现某个 DeFi 协议的 TVL 在 24 小时内涨了 50%，写一条推特

**回答：**
```
Unbelievable, uniswap v4 TVL increaed 50% in last 24 hours. 
Follow me to find out which tokens raised this.
```

**分析：**

| 优点 | 缺点 |
|------|------|
| ✓ 有 hook（Unbelievable） | ✗ increaed → increased（拼写） |
| ✓ 有 CTA（Follow me） | ✗ in last → in THE last（冠词） |
| ✓ 结构正确 | ✗ raised this → drove this（用词） |

**修正版：**
```
Unbelievable - Uniswap v4 TVL increased 50% in the last 24 hours.
Follow me to find out which tokens drove this.
```

---

### 1B. 长写作

**题目：** Why do you want to build a Web3 product?

**回答：**
```
I think web3 is permissionless, anyone can build any product on this. 
Nobody can shut it down. And this is earyly window for web3, there are 
many oppotunities amount this. I want to be rich from this.
```

**分析：**

| 错误 | 修正 |
|------|------|
| earyly | early |
| oppotunities | opportunities |
| amount this | around this / in this space |
| on this | in this space / on it |

**修正版：**
```
I think Web3 is permissionless - anyone can build any product in this space. 
Nobody can shut it down. And this is an early window for Web3, there are 
many opportunities around this. I want to get rich from this.
```

## Part 3: Vocabulary 详情

### 3A. 词汇选择

| 题目 | 回答 | 正确答案 | 结果 |
|------|------|----------|------|
| The token price _____ by 20% | raised | rose | ✗ |
| higher TVL _____ its competitors | than | than | ✓ |
| The team is _____ on a new feature | working | working | ✓ |
| do your own research _____ investing | before | before | ✓ |
| The hack _____ in a loss of $50M | resulted | resulted | ✓ |

**重点错误：raise vs rise**
- raise：及物动词，需要宾语（raise something）
- rise：不及物动词，主语自己上升（price rises）
- 正确：The price **rose** by 20%
- 错误：The price **raised** by 20%

### 3B. 词汇解释

| 词汇 | 回答 | 评价 |
|------|------|------|
| TVL | Total value locked | ✓ 正确 |
| Whale | someone who has a lot money | △ a lot → a lot of |
| Airdrop | an acitivity that distribute tokens to many people freely | △ acitivity → activity, distribute → distributes |

## Part 4: Grammar 详情

### 4A. 语法改错

| 原句 | 回答 | 正确答案 | 结果 |
|------|------|----------|------|
| The protocol have been audited | have → has | have → has | ✓ |
| one of the best trader | is → was | trader → traders | ✗ |
| I am interesting in DeFi | interesting → interested | interesting → interested | ✓ |
| they will launched | will → would | launched → launch | ✗ |
| more better | more → much | more better → much better / better | ✓ |

**重点错误：**
1. **one of the + 最高级 + 复数名词**
   - one of the best traders ✓
   - one of the best trader ✗

2. **will + 动词原形**
   - will launch ✓
   - will launched ✗
